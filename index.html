Introduction

The Mapzen Targeted Editing series consists of a set of blog posts designed to encourage specific edits to OpenStreetMap data.

Getting Started

Feel free to start by cloning the repo. Each Targeted Editing post consists of a set of SQL, yaml, and JavaScript.

The following steps, designed for those working locally, outline the basic workflow associated with the queries and maps built for Targeted Editing blog posts.

  • Develop a Theme:

    This step includes tag research, and metro area choices.

  • Download Metro Extracts:

    This step employs a script utilizing wget to download multiple Metro Extracts.

  • Build Databases:

    This step employs a script to build databases locally.

  • Query Data:

    This step generates basic summary statistics for each metro area.

  • Customize Tangram Tiles:

    This step customizes Tangram tiles to highlight desired features with custom colors and/or custom labels.

  • Build Map Interactivity:

    This step employs html and JavaScript to link highlighted features to OpenStreetMap editing environments. View Map Gallery.
